# SVG Converter App

A Electron + Vue 3 desktop app to convert SVG files to other formats like PNG, ICO, ICNS, etc.

## Project Setup

```sh
npm install
```
## Compile and Hot-Reload for Development

```sh
npm run dev
```

## Type-Check, Compile for Production

```sh
npm run build
```

## Type-Check, Compile, Package for Production

```sh
npm run release
```
